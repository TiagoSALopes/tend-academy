# TEND Academy

Sistema inteligente para gestão de rotinas de estudo.

## Funcionalidades
- Importação automática de horários via Excel (.xlsx).
- Extração de disciplinas por turma (ENI/ENC).
- Geração de plano de estudos personalizado com IA.
\
## Tecnologias
- PHP 8.x
- JavaScript (Fetch API)
- Tailwind CSS
- PhpSpreadsheet

## Como instalar
1. Clona o repositório: `git clone [link-do-repo]`
2. Instala as dependências: `composer install`
3. Configura a base de dados no ficheiro `Core/Database.php`.
